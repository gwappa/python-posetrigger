Installation
------------

If you use the 64-bit Intel CPU, you can use the ``FastEventServer_linux_x86-64`` provided in the ``libraries`` directory of the repository:

   .. code-block:: Bash

       $ sudo cp FastEventServer_linux_x86-64 /usr/local/bin/fasteventserver
